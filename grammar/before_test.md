* 문자열을 하나씩 나누어 리스트로 정리하기
```
a = '12345'
list(a) // ['1', '2', '3', '4', '5']
```

* 문자열 뒤집기
  - [::-1]
```
a = '734 893'
a[::-1] // '398 437'
```
* 2. 리스트 뒤집기
   - reverse()
```
a = '734 893'
a = list(a) // ['7', '3', '4', ' ', '8', '9', '3'] // 변수 지정 필요
a.reverse // ['3', '9', '8', ' ', '4', '3', '7'] ## 변수 변환 필요 없음
result = '.'join(a) // '398 437'
```

* 단어 변환
```
a = 'abc'
a = a.replace('a', '*')
print(a) ## '*bc'
```
